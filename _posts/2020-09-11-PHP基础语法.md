> 基本语法

## 定义
- php文件先解析在执行（Apache）,双击打开文件显示是源代码，域名打开才是结果

---

## 基本语法
- <?php (常用)没有结尾。常用标记，只能写php代码
- <?php ?> 完整写法，可嵌套HTML
- <? ?> 短标记，需要修改Apache配置  shprt_open_tag = On
- <% %> 新版已删除
- 空白字符:空格，回车，制表符（tab）,不影响代码的执行，用于排版。

---

## 注释快捷键
- 单行         #
- 多行         /*  */
- 单行快键键   ctrl+/
- 多行注释     alt+shift+a

---

## 变量
- $ 变量标识符。可以使用字母，数字下划线。
- 不能数字开头，中文不推荐。
- 引用变量【常用】$b = &$a   变量a的值赋给变量b

```php
    $a = "b";
    $b = "1"; 
    var_dump($$a); //1
```

> 数据类型

## 四大标量
- int
- string
- bool
- float

---

## 字符串的使用
- 只能双引号解析变量：
- $a = "今天天气怎么样! {$b}";
- 只能双引号解析特殊字符
- \n 换行
- \r 回车
- \t tab键 制表符

```php
     $a = "可以解析特殊字符: \t\r\n!";
```

---

## 字符串嵌套
- 使用转义符 \

```php
     $a = "你\"好\"";
```

---

## 定义长篇文章
- 定界符<<<CTO ... CTO

```php
    $str = <<<CTO 
    内容 
    CTO;
```

---

## 复合结构类型
- 数组类型 array()
- 对象类型 object()

```php
     $var = array();
     $var = new Model();
```

---

## 特殊类型
- 资源类型 resource
- 空 null

```php
     $d = fopen('1.php','d');  // fopen('1.php','添加的内容')
```
